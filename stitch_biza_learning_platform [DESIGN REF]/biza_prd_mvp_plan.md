# BIZA - Full Product Documentation Suite
Version 1.0 · Confidential · 2025

## 1. Product Requirements Document (PRD)
### 1.1 Product overview
**Product name:** Biza
**Tagline:** Learn it. Earn it.
**Type:** Online education platform for legitimate internet income
**Primary market:** Kenya (ages 18–30)

### 1.3 Solution
Biza provides verified, structured roadmaps for every major legitimate online income category. Each course follows a fixed 7-step framework: what it is, requirements, how money flows, the action process, platforms, curated video resources, and scam warnings.

## 2. MVP Plan
### 2.2 MVP feature scope
- User registration & login (P1)
- 6 categories with 3 courses each (P1)
- 7-step roadmap per course (P1)
- Step locking (free vs Pro) (P1)
- M-Pesa payment (STK Push) (P1)
- User dashboard with progress (P1)
- Mobile responsive design (P1)
